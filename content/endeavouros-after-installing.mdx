## Edit Scaling

Open file `~/.Xresources` and add this line

```bash
Xft.dpi: 125
```

Save file and run command and reboot 

## Install Bluetooth Manager

Run these command

```bash
sudo systemctl start bluetooth
sudo systemctl enable bluetooth
sudo pacman -S --needed bluez bluez-utils
sudo pacman -S blueman
```

**Fix audio not export bluetooth device**

Create file at `/etc/pulse/default.pa` and enter content: 

```bash
### Automatically switch to newly-connected devices
load-module module-switch-on-connect
```

Restart device

## Install Ibus-bamboo

```bash
sudo pacman -S ibus

mkdir ibus-bamboo
cd ibus-bamboo
wget "https://raw.githubusercontent.com/BambooEngine/ibus-bamboo/master/archlinux/PKGBUILD-git" -O PKGBUILD
makepkg -si
```

Add thess lines to `/etc/environment`

```bash
GTK_IM_MODULE=ibus
QT_IM_MODULE=ibus
XMODIFIERS=@im=ibus
```

Run this command to autostart Ibus-bamboo

```bash
ibus-daemon -rxRd
```

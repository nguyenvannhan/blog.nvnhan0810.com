## Install Alacritty

```bash
sudo pacman -S alacritty
```

## Edit Scaling

Create file `~/.Xresources` and add this line

```bash
Xft.dpi: 192
```

Open file `~/.xinitrc` and add:

```bash
xrdb -merge ~/.Xresources
```


Save file and run command and press: `Win + Shift + r` 

## Config i3

Add these configs to `~/.config/i3/config`

```bash
bar {
    status_command i3status
    position top
}

for_window [class="^.*"] border none

gaps inner 10px
```

## Install microcode update

```bash
sudo pacman -S intel-ucode
```

## Install Bluetooth

Run these command

```bash
sudo pacman -S --needed bluez bluez-utils
sudo modprobe btusb
sudo systemctl start bluetooth
sudo systemctl enable bluetooth
sudo pacman -S blueman
```

## Install Ibus-bamboo

```bash
sudo pacman -S ibus

bash -c "$(curl -fsSL https://raw.githubusercontent.com/BambooEngine/ibus-bamboo/master/archlinux/install.sh)"
```

Add thess lines to `/etc/environment`

```bash
GTK_IM_MODULE=ibus
QT_IM_MODULE=ibus
XMODIFIERS=@im=ibus
```

Add this line to `~/.xprofile`

```bash
ibus-daemon -rxd
```

## Clone dotfile

```bash
git clone git@github.com:nguyenvannhan/dotfile.git
pacman -S eza
```

### Install Nerd Fonts

```bash
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/JetBrainsMono.zip
mkdir jetbrains-mono-nerd-fonts/
sudo pacman -S unzip
unzip JetBrainsMono.zip -d jetbrains-mono-nerd-fonts/
sudo mv jetbrains-mono-nerd-fonts/ /usr/share/fonts/
sudo fc-cache -v
```

## Update config Alacritty

```bash
ln -s /home/nvnhan/dotfile/configs/alacritty/ /home/nvnhan/.config/
```

## Config ZSH

```bash
# Install ZSH
sudo pacman -S zsh
sudo chsh -s $(which zsh)
ln -s /home/nvnhan/dotfile/configs/.zshrc /home/nvnhan/.zshrc
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
source ~/.zshrc
```

## Config fnm

```bash
curl -fsSL https://fnm.vercel.app/install | bash -s -- --install-dir "./.fnm" --skip-shell
fnm install --lts
```

## Config tmux

```bash
sudo pacman -S tmux
ln -s /home/nvnhan/dotfiles/configs/tmux /home/nvnhan/.config/tmux
sudo rm -r ~/.config/tmux/plugins
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```
Use `Ctrl + Shift + I` to install extension for tmux
Use `Ctrl +b` and `r` to reload config tmux

## Install Visual Studio Code
```bash
cd /opt
sudo git clone https://aur.archlinux.org/yay-git.git
sudo chown nvnhan:nvnhan yay-git
cd yay-git
makepkg -si
yay -S visual-studio-code-bin
```

## Setup Theme
```bash
sudo pacman -S lxappearance-gtk3 arc-gtk-theme dracula-icons-theme
yay -S dracula-icons-theme
```

Open `lxappearance` and setting theme and icon

Open file `~/.config/gtk-3.0/settings.ini and add lines
```bash
gtk-application-prefer-dark-theme=1
```
